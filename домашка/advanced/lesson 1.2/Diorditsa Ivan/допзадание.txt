домашка (это вместо второго дополнительного) - мини инстаграм

у вас есть TCP сервер на который можно по HTTP залить фоточку, скачать фоточку, удалить фоточку
и у вас есть HTTP клиент для вашего мини инстаграмма, через который можно это сделать
реализуйте коды ошибок (например если попытаться удалить фоточку, которой на сервере не существует)

используйте socketserver
для реализации HTTP используйте urllib и requests (например сервер на requests, клиент на urllib)
нельзя использовать модуль http.server или любой другой модуль, где реализация http уже сделана (подолбайтесь)
